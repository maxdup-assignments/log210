Log210
====
 
Installation
====
install python 2.7 ("sudo apt-get install python2")
install pip2 ("sudo apt-get install pip2")

if using virtualenv
    "virtualenv env"
    ". env/bin/activate"

install dependencies (pip install -r requirements.txt)

Running the server
====
move to log210/application
"python2 manage.py runserver"
